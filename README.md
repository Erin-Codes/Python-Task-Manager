# CapStone-IV-L1-
This Program is an extended version of the **task_manager** program in the previous CapStone repositories.
## Added features:
* The *reg_user* option now ensures that user names aren't duplicated.
* Tasks are displayed in a more user-friendly manner.
  * Users can now edit their tasks and or mark it as complete.
* New *generate reports* option added to the main menu.
  *  These reports contains *user overview* and *task overview*.
